HackerNews Follow Discussion Button

Hello, this is a small CSS button for those who like to insert a link to a HN Discussion that relates their product.

    Welcome to the first version of the HackerNews Follow Discussion Button.

    Basically, this is plain vanilla. Just a .html and .css file.

    What can the Button be used for? I thought it a nice way for young startups or upcoming YC batches to setup a small link to their website related discussion on HackerNews (news.ycombinator.com). This is thought to be an alternative approach to the HN Submit and Counter Button (see https://github.com/igrigorik/hackernews-button). Alternative, as it only serves as a link to the discussion, that´s it.
